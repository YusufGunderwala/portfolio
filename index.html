Jarvis, Your Personal Assistant

Project Link
Category
Python Development
Client
Personal Project

Project Description

Jarvis is a versatile virtual assistant project that integrates voice recognition and natural language processing to perform a range of tasks. Developed in Python, Jarvis can handle various commands including opening websites, fetching news based on dynamic categories, and launching applications. The assistant uses multiple libraries and APIs to deliver a seamless experience. Features

  • Voice Recognition: Uses the speech_recognition library to listen to and process user commands.
  • Text-to-Speech: Employs pyttsx3 for converting text responses into speech, providing an interactive experience.
  • Web Browsing: Opens specified websites such as Google, Facebook, YouTube, and LinkedIn.
  • Dynamic News Fetching: Retrieves and reads out news headlines based on dynamically extracted categories from user commands. Integrated with the NewsAPI for up-to-date news.
  • Application Launching: Opens specified applications dynamically. Supports commonly used applications and can be extended to include more.
